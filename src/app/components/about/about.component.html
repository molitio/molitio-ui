Description of molitio

What is molitio and what is not.

molitio is intended to be an open source resource distribution engine with live topographical fead back, sort of data displayed on a map.

Two main types of resources recognized by the system,

  • desideratum (old latin for desire)
  • contribution

subcategories are a breakdown of the given resource type.

it aims to match desired and only desidered needs of human beings with contributions of other human beings contributions would be regarded with a single unit of "Token Of Good Deed" or "Good Deed Token" so high contributors can be recognized.

the project is at it's infant phase lots of details regarding the functionalty to be yet implemented

molitio is not a service thats intended for any monetary gain, it is open source and should be crowd sourced in the regards of digital compute resource needs
